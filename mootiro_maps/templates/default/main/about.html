{% trans "Understand a territory by its map" %}

{% blocktrans %}MootiroMaps helps to make an online map of your community that shows where resources, demands and organizations are located.{% endblocktrans %}

{% trans "Beautiful and informative maps" %}

{% blocktrans %}Colours and icons make your map easy to understand and navigate{% endblocktrans %}

{% trans "Instant information" %}

{% blocktrans %}Passing your mouse over the mapped objects you might access brief information about each content{% endblocktrans %}

{% trans "Search the map" %}

{% blocktrans %}You can define a radius and search for all the objects contained in this area{% endblocktrans %}

{%trans "Rich information about each object on the map" %}

{% blocktrans %}Each object on the map has its own page with contact information, detailed description, tags and a lot more.{% endblocktrans %}

{% trans "Tell a story" %}

{% blocktrans %}Add photos, logos and link to other content on the map{% endblocktrans %}

{% trans "Help others to find your data" %}

{% blocktrans %}Define tags for your data and make it easy for others to search your content{% endblocktrans %}

{% trans "Show relations" %}

{% blocktrans %}Relate contents to each other and promote transparency! Show, for instance, that a foundation is working in more than one community or which organizations received resources from the same donor.{% endblocktrans %}

{% trans "MootiroMaps supports your mapping projects - you might reuse data mapped by other users, import data from spreadsheets and engage others to map collaboratively." %}

{% trans "Your space on MootiroMaps" %}

{% blocktrans %}Your mapping project has its own page on MootiroMaps. You might present your initiative, explain workflow and planned actions and show how other users can collaborate.{% endblocktrans %}

{% trans "Add collaborators" %}

{% blocktrans %}Add users to you project map that can edit your map and add new content.{% endblocktrans %}

{% trans "Get to know similar maps" %}

{% blocktrans %}Based on the tags related contents will be shown on the page of your mapping project.{%endblocktrans %}

{% trans "Give visibility to your partners" %}

{% blocktrans %}Upload logos of the partners of your project{% endblocktrans %}

{% trans "Reuse data from the map" %}

{% blocktrans %}Add contents that have been created by other users to your mapping project{% endblocktrans %}

{% trans "Import data to your map" %}

{% blocktrans %}Integration with Google Spreadsheets and Google Fusion Table allows you to import data into your project. You can clean data that you scraped from the internet or received through freedom of information requests, simulate the import and finally upload to your project.{% endblocktrans %}

{% trans "By analyzing the territory and location of services you will get import insights for community planning and and better understand priorities and demands for local development." %}

{% trans "Map local needs and discuss with the community" %}

{% blocktrans %}With MootiroMaps you can map the demands of a community and invite its residents and local organizations to discuss and find solutions. {% endblocktrans %}

{% trans "Monitor progress of community projects" %}

{% blocktrans %}Document ongoing projects uploading reports and photos{% endblocktrans %}

{% trans "Analyze the mapped data" %}

{% blocktrans %}By mapping public services and local organizations you will easily understand which parts of the community are well served and where new social services should be created.{% endblocktrans %}
